SETTING
1) Comando slash > Set Weight @role <value>
[Per ogni ruolo viene assegnato un moltiplicatore al voto]

2) Comando slash > Set votanti: @role, @role, @role
[Solo i ruoli selezionati possono votare]

3) Comando slash > set expiration > Passed with  % dei partecipanti votanti
[La votazione viene approvata solo se viene superata la % richiesta]

4) Comando slash > set countdown >  (minutes)
[comando opzionale in caso si vuole settare un limite di tempo]

5) Comando slash > kill votazione
[Comando opzionale se si vuole annullare la votazione]

6) Comando slash > set result channel #channel
[canale dove si condivide il risultato della votazione]

7) Comando slash > mostra i ruoli che votano e i numeri di persone per ogni ruolo che votano > Si o no
[Per scegliere se fare vedere quante persone hanno votato e con quale ruolo (moltiplicatore)]

Avvio votazione

7) Comando slash > vote > question:  > Choice A: Yes, Choice B: Maybe, Choice C: No
[Embed con reazioni come simple poll (automaticamente il bot mette 1 reazione di default in base alle Choice inserite]
